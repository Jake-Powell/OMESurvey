# Survey values

A list of vectors where each vector contains strings representing survey
answers to a single question. This includes:

- 'Agree a lot', 'Agree a little', 'Neither', 'Disagree a little',
  'Disagree a lot'

- 'Almost always','Most of the time','Half the time','Some of the
  time','Almost never'

- 'Often', 'Sometimes', 'Never'

- 'A lot', 'Sometimes', 'Not much', 'Did not answer'

- 'Good', 'Okay', 'Bad', 'Did not answer'

- 'Like', 'Okay', "Don't like", 'Did not answer'

- '1','2'

## Usage

``` r
survey_values
```

## Format

An object of class `list` of length 10.
