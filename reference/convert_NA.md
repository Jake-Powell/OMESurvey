# Convert "N-A" to NA across a data frame

Convert "N-A" to NA across a data frame

## Usage

``` r
convert_NA(data)
```

## Arguments

- data:

  a data frame

## Details

Loops over all columns and changes "N-A" to NA.

## Examples

``` r
if (FALSE) {
OMESurvey::survey_example |> convert_NA()

}
```
